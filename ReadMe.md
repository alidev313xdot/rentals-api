# 🏠 Rentals System — My First .NET Project

## 📌 Project Overview

This project is a backend system built using ASP.NET Core to help people in Karachi find affordable, clean, and reliable rental properties from trusted property owners.

What started as a simple API project is now evolving into a real-world system focused on solving a local problem: **making rental discovery easier, safer, and more transparent for short-term visitors and residents.**

---

## 🌱 Current Progress (Phase 1)

We have successfully built the initial foundation of the system with basic APIs:

### ✅ Implemented Features

* Get all properties
* Get property by ID
* Add a new property
* Search properties based on:

  * Area
  * Number of rooms
  * Furnished / Unfurnished
  * Verified properties

These APIs serve as the **first step toward a complete rental ecosystem**.

---

## 🎯 Core Vision

The main goal of this system is to:

* Help users find **low-cost rentals**
* Ensure properties are **clean and well-maintained**
* Promote listings only from **trusted / verified owners**
* Provide a **fast and reliable search experience**
